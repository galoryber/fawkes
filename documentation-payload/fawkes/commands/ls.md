## Summary

List files and folders in a directory. Defaults to the current working directory. Integrates with the Mythic file browser.

### Arguments

#### path (optional)
Directory path to list. Defaults to current working directory.

## Usage
```
ls [path]
```

Example
```
ls
ls C:\Users\admin\Desktop
ls /var/log
```
