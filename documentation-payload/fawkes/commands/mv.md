## Summary

Move or rename a file from source to destination.

### Arguments

#### source
Path to the source file.

#### destination
Path to the destination.

## Usage
```
mv <source> <destination>
```

Example
```
mv C:\Temp\payload.exe C:\Temp\svchost.exe
mv /tmp/output.log /tmp/output.log.bak
```
