## Summary

Copy a file from source to destination.

### Arguments

#### source
Path to the source file.

#### destination
Path to the destination file.

## Usage
```
cp <source> <destination>
```

Example
```
cp C:\Users\admin\file.txt C:\Temp\file.txt
cp /etc/hosts /tmp/hosts.bak
```
