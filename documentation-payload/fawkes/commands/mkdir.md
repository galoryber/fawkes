## Summary

Create a new directory. Creates parent directories if they don't exist.

### Arguments

#### directory
Path to the directory to create.

## Usage
```
mkdir <directory>
```

Example
```
mkdir C:\Temp\staging\payloads
mkdir /tmp/exfil
```
